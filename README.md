# 🏎️ F1 待办事项 Todo List

一个以 Formula 1 赛车为主题的待办事项应用，使用 React + Tailwind CSS 构建，数据存储于 localStorage。

## 功能特性

- ✅ 添加 / 完成 / 删除任务
- 🔍 按状态筛选（全部 / 待完成 / 已完成）
- 💾 localStorage 持久化存储
- 🏎️ F1 赛车图片背景滚动播放（每 3 秒切换）
- 💬 完成任务时弹出 F1 主题鼓励语
- 🎯 任务优先级（高优先级 / 普通）
- 📅 截止日期 + 过期自动标红
- 🖱️ 拖拽排序（基于 dnd-kit）

## 技术栈

- React 19 + Vite 8
- Tailwind CSS v4
- @dnd-kit/core + @dnd-kit/sortable
- localStorage

## 本地运行

```bash
npm install
npm run dev
```

## 更新记录

### 2026-03-15

**新增：任务优先级**
- 每个任务新增 `priority` 字段，支持 `high` / `low`
- 创建任务时可选择“普通”或“高优先级”
- 高优先级任务以红色文字和标签高亮显示

**新增：截止日期**
- 创建任务时可设置截止日期
- 每个任务会显示截止日期信息
- 未完成且超过截止日期的任务会标红，并显示“已过期”

**升级：拖拽排序改用 dnd-kit**
- 替换原生 HTML5 drag API，改用 `@dnd-kit/core` + `@dnd-kit/sortable`
- 通过专用手柄（≡）触发拖拽，避免误触 checkbox / 删除按钮
- 拖拽激活距离设为 5px，防止点击时误触发
- 支持鼠标和触摸屏
- 在”全部 / 待完成 / 已完成”筛选视图下均可重排，顺序持久化到 localStorage

**兼容性处理**
- 旧的本地任务数据会自动补齐 `priority` 和 `dueDate` 默认值

### 2026-03-14

**新增：F1 背景图轮播**
- 使用 6 张 F1 赛车主题图片（来源：loremflickr.com）
- 每 5 秒自动切换，带淡入淡出过渡动画
- 添加半透明黑色遮罩，保证内容可读性

**新增：鼓励语功能**
- 内置 8 条 F1 主题中文鼓励语
- 每次勾选完成任务时，随机弹出一条鼓励语
- 鼓励语以黄色气泡形式展示，3 秒后自动消失

**UI 优化**
- 整体风格从浅色渐变改为赛车主题深色毛玻璃卡片
- 主色调改为 F1 赛旗黄（yellow-400）
- 标题增加 F1 赛车 Emoji 和副标题
